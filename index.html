Server Ranks

Climb the ladder and earn your place in the Villa SMP community.

PLAYER
🟢

Player

The default rank every member starts with. Begin your survival journey here.

  • Access to basic commands
  • Set up to 3 homes
  • Participate in events
  • Use /spawn, /warp, /tpa
  • Access to public shops
SUPPORTER
💎

Supporter

Members who support the server and unlock extra perks and quality-of-life features.

  • Up to 8 homes
  • Colored chat name
  • Access to /hat and /nick
  • Priority queue access
  • Monthly crate keys
MODERATOR
🛡️

Moderator

Trusted staff who keep the community safe and enforce the rules fairly.

  • Staff chat access

  • Ticket management

// Getting Started

Server Guide

New to Villa SMP? Follow these steps to get started.

1

How to Join

Open Minecraft Java Edition 1.8.8, click "Multiplayer", then "Add Server". Enter the server address VillaSmp.eagler.host and click "Done". Join and start playing!

2

How to Start

Once you spawn, read the rules at spawn, grab your starter kit, and venture out into the wild. Use /rtp to teleport to a random location and begin building your base.

3

Important Commands

Learn these essential commands to navigate and enjoy the server:

/spawnTeleport to spawn
/warpOpen warp menu
/helpSee all commands
/tpaRequest teleport
/homeTeleport home
/sethomeSet a home
/moneyCheck balance
/msgPrivate message
